Frequently Asked Questions

What is Sentira AI?

Sentira AI is an advanced web application that analyzes speech to detect stress and emotional states using deep learning models.

Which emotions can it detect?

The system can detect emotions such as Happy, Sad, Angry, and Neutral, along with stress levels.

What file formats are supported?

Currently, only WAV audio files are supported for upload and analysis.

Is my data private?

Yes, your data is processed securely and is not stored after analysis.
